# Simple Blog Platform (frontend-only)

This is a minimal blog demo using HTML/CSS/JS that runs entirely in the browser and stores data in `localStorage`.

Features:
- Register / Login (no security — demo only)
- Create and delete posts (author only)
- Comment on posts

How to run:
1. Open `index.html` in your browser.
2. Click "Login / Register" to create a user or login.
3. Create posts and add comments.

Notes:
- This is a demo for learning; do NOT use in production. Passwords are stored in plain text in localStorage.
- To clear demo data: double-click the header and confirm.
